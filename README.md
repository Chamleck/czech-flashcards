# Чеські картки — Фаза 1

Застосунок для вивчення чеської мови через флеш-картки (укр ↔ чес) з повним
відмінюванням іменників за 7 відмінками, родами та числами.

**Стек:** React Native + Expo + TypeScript.

## Що вже є

**Фаза 1**
- 🔤 **Слова** — 19 іменників, кожен покриває один з 11 базових зразків відмінювання
  (pán, muž, hrad, stroj, žena, růže, kost, město, moře, kuře, stavení). Картка: укр слово →
  показати чеську + повну таблицю 7×2 + приклад речення.
- 📚 **Граматика** — відмінки, роди й зразки з поясненнями.
- Інтервальне повторення (spaced repetition) зі збереженням прогресу локально.
- Кольорове кодування за родом.

**Фаза 2**
- 🧭 **Навігація** на React Navigation (native stack) — працює апаратна кнопка «назад».
- 📱 Коректні відступи під системну шторку/жести (`react-native-safe-area-context`).
- 🗂️ **Категорії слів** (Люди, Дім і побут, Їжа, Місто, Транспорт, Природа) — можна обрати
  всю категорію або окремі слова перед сесією.
- 🔁 Авто-колода **«Повторити помилки»** — слова, позначені «не знаю», збираються окремо.
- 📚 **Граматика за темами** з розширюваною моделлю контенту (рід/число, відмінки, зразки;
  далі — дієслова, займенники, прийменники, числівники).
- 🧱 Таблиця відмінювання перебудована (стек-layout) — довгі форми більше не ламають рядок.

## Що попереду (наступні фази)

Прикметники → займенники, прийменники → дати/місяці → фрази → речення з пропусками.

## Roadmap: локалізація (фінальний етап)

Російська локалізація (інтерфейс + картки + граматика, перемикач укр/рос на головному
екрані) — зафіксована вимога на майбутнє. Реалізовувати **тільки в самому кінці**, після
повного наповнення контентом усіх фаз. Не робити паралельно з основною розробкою —
кожне нове слово/тему довелося б одразу перекладати двічі, що вдвічі уповільнить наповнення.

## Як зібрати APK (без Android Studio)

Збірка відбувається у хмарі Expo (EAS Build). Потрібен безкоштовний акаунт.

### 1. Встанови інструменти (одноразово)
```bash
npm install -g eas-cli
```

### 2. У папці проєкту
```bash
npm install
```

### 3. Увійди в акаунт Expo
```bash
eas login
```
(Якщо акаунта нема — зареєструйся на https://expo.dev, це безкоштовно.)

### 4. Прив'яжи проєкт
```bash
eas init
```
Це створить `projectId` і впише його в `app.json` автоматично
(можна прибрати рядок `"projectId": "PUT-YOUR-EAS-PROJECT-ID-HERE"` — він перезапишеться).

### 5. Запусти збірку APK
```bash
eas build -p android --profile preview
```
Профіль `preview` (див. `eas.json`) збирає саме `.apk` (не `.aab`), який можна одразу
поставити на телефон.

### 6. Скачай файл
Коли збірка завершиться (кілька хвилин), EAS дасть посилання — завантаж `.apk`, перекинь на
телефон, дозволь встановлення з невідомих джерел, постав.

## Локальний запуск для перевірки (опційно)

```bash
npm install
npx expo start
```
Скануєш QR-код застосунком **Expo Go** на телефоні — застосунок відкриється одразу, без збірки.
Зручно для швидкого тесту перед фінальною збіркою APK.
